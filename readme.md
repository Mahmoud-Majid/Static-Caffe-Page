# Caffe shop

This a task for an eT3 Internship 2022

In this project I used React, plain CSS for styling and Bootstrap icons.

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
